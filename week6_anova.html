MLR: ANOVA Table
Analysis

Deconstructing Excel's regression output: tracking statistics, degrees of freedom, hypothesis testing, and coefficient intervals.

01

Summary Regression Statistics

The first block of any standard MLR output summarizes the model's overall explanatory power and general error margins.

Multiple R

The explicit correlation coefficient between the actual $Y$ values and the model's predicted $\hat{Y}$ values.

R Square ($R^2$)

Coefficient of Determination. Indicates the exact percentage of total variation in $Y$ that can be explained by the $X$ variables.

Standard Error ($SE$)

The Root Mean Square Error (RMSE). It shows, on average, how far off the predictions are from the actual values.

$SE = \sqrt{MS_{Residual}}$

Observations ($n$)

The total sample size (number of data points/rows) used to train the regression model.

02

ANOVA Table Anatomy

Analysis of Variance (ANOVA) explicitly partitions the total variation into "Explained" (Regression) and "Unexplained" (Residual) segments to calculate the test statistics. Let $k$ equal the number of independent variables.

03

Overall Model Significance (F-Test)

The ANOVA table's primary purpose is to test if the entire model is statistically valid. It generates the Significance F (p-value) derived from the F-Statistic.

Null Hypothesis ($H_0$)

The model is NOT significant. All slope coefficients are exactly zero.
$\beta_1 = \beta_2 = \dots = \beta_k = 0$

Alternate Hypothesis ($H_1$)

The model IS significant. At least one slope coefficient is distinct from zero and offers predictive power.

The Rejection Rule

If Significance F (p-value) < $\alpha$ (typically 0.05), we reject $H_0$ and conclude the model is valid.

04

Individual Predictor Significance (t-Test)

Even if the overall model is valid, individual variables might be useless. The Coefficients Table runs an independent t-test for the intercept and every single $X_i$ variable. The Null Hypothesis for each row is that its specific $\beta_i = 0$.

Estimate (Coefficient)

The calculated $\beta$ value that will be plugged directly into the $Y = \dots$ equation.

Standard Error

The uncertainty or margin of error strictly concerning this specific coefficient estimate.

t-Statistic

$\frac{\text{Estimate}}{\text{Std Error}}$

If the P-value in a variable's row is $> 0.05$, we fail to reject $H_0$. That variable does not add statistically significant predictive power alongside the others, and it should generally be removed from the model.

05

Confidence Intervals & Model Building

The final columns of the output establish the Lower 95% and Upper 95% boundaries for the true coefficient value. This provides a visual alternative to the p-value.

The Zero-Crossing Rule

If the 95% interval for a variable spans from a negative number to a positive number (e.g., Lower: -0.011 to Upper: +0.011), it means $0$ is a highly probable true value for that coefficient.

Crossing zero is mathematically identical to having a P-value $> 0.05$. The variable is insignificant.

Extracting the Final Equation

Once you drop the insignificant variables (those with high p-values or zero-crossing intervals), you build your predictive equation using the remaining "Estimates":

$Y_{pred} = (\text{Intercept Estimate}) + (\text{Estimate}_1 \cdot X_1) + (\text{Estimate}_2 \cdot X_2)$
